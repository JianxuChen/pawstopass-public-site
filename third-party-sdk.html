Current public release dependencies

First-party telemetry service

  • Purpose: anonymous or pseudonymous browser telemetry and gameplay reporting
  • Current identity scope: session_only in the current itch release telemetry configuration; no cross-session telemetry player_id is stored locally by default
  • Current endpoint family: marble-drop-telemetry*.workers.dev
  • Current data path: browser telemetry client -> Cloudflare Worker -> Cloudflare D1

Cloudflare Workers + D1

  • Purpose: first-party event ingest, storage, and aggregation
  • Service-layer note: Cloudflare may use necessary service cookies, network logs, and related infrastructure identifiers for service security, delivery, and logging
  • Implementation family: Worker ingest endpoint plus D1 event and aggregate tables

Godot / Godot Mono runtime

  • Purpose: game runtime and C# execution
  • Engine version used by this project: Godot 4.6
