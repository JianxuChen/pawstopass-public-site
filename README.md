# Public Site

These static files are ready to be published through any simple static host such as GitHub Pages, Cloudflare Pages, Netlify, or a plain object-storage website.

Recommended public paths:

- `/privacy-policy.html`
- `/terms.html`
- `/support.html`
- `/third-party-sdk.html`
- `/resources-licenses.html`

After hosting:

1. use the final hosted URLs in the current `itch-web` public release page or linked disclosure surfaces
2. keep the hosted privacy, terms, support, and third-party pages aligned with the current `itch-web` runtime scope
3. if the hosted URLs change, update any `itch-web` release-facing docs or page links that reference them
4. verify all links open on desktop and mobile browsers before publish
