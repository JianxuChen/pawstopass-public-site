Paws To Pass / 鼠你能过

Updated: 2026-05-17 · Scope: CrazyGames HTML5 build

Current disclosed services

  • First-party telemetry service: anonymous or pseudonymous browser telemetry and gameplay reporting
  • Cloudflare Workers + D1: event ingest, storage, and aggregate reporting
  • CrazyGames SDK: platform lifecycle, wrapper integration, settings mute state, and SDK data integration

Current identity scope

The current CrazyGames telemetry posture uses session_only identity. It does not keep a cross-session telemetry player_id in local browser storage by default.

Current Basic Launch exclusions

  • external ad SDKs
  • external login SDKs
  • social-sharing SDKs
  • user-identity analytics beyond the first-party telemetry path described above

Consistency rule

Before submitting a CrazyGames candidate with telemetry enabled, this page, the CrazyGames privacy policy text, the actual runtime telemetry configuration, and the deployed Worker/D1 environment must all describe the same behavior.
